# Replication Materials: Energy Consumption Timber Blockchain

This repository contains the full replication materials for the study on energy
consumption in timber-based blockchain systems.

## Data
The main replication file is:

- `data/Energy_Consumption_BC_Calculation.xlsx`

The Excel file contains all data sources, assumptions, and calculations required
to reproduce the results reported in the paper.

## Reproducibility
All results reported in the associated publication can be reproduced directly
from the provided Excel file without the need for additional software beyond a
standard spreadsheet application.
